====================

コミットする前に空白をチェックして、見つけると強制的にコミットをさせなくするヤバいフック

つかいかた
-------------------

`(リポジトリ)/.git/hooks/pre-commit` にリネームして置いて、
`chmod +x pre-commit` とかで実行権限を与える
